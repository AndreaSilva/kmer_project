The project contains files for Dubinkina et al, "Assessment of k-mer spectrum applicability for metagenomic dissimilarity analysis".
Contents of the subfolders:

- data: additional files with reference methods output and k-mer counts could be obtained from http://download.ripcm.com/Dubinkina_2015_suppl_data/data/

- output: graphics and distance matrices obtained during analysis

- scripts: processing scripts and additional data.
